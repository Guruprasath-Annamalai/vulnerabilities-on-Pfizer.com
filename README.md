# vulnerabilities-on-Pfizer.com
This project demonstrates the identification of security vulnerabilities using Burp Suite, including CORS misconfiguration, XPath Injection, and TLS certificate misconfiguration, along with their potential security impact.  
# Pfizer Web Security Assessment

## Overview

This repository contains the results of an automated security assessment performed using Burp Suite Professional against publicly accessible web resources.

The objective of this assessment was to practice vulnerability identification, understand Burp Suite's automated scanning capabilities, and analyze reported findings.

> Disclaimer:
> This repository is for educational and portfolio purposes only.
> The reported findings are generated by automated scanning and have not been manually validated. Automated scanners can produce false positives. No exploitation was performed.
